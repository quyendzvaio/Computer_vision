Edge Device — Capture & Send

Edge device:
  Script Python đơn giản (ko cần PyQt/PySide)

  main():
    cap1 = cv2.VideoCapture(0)   # USB Cam1
    cap2 = cv2.VideoCapture(2)   # USB Cam2
    ctx = zmq.Context()
    pub1 = ctx.socket(zmq.PUB)   # port 5555
    pub2 = ctx.socket(zmq.PUB)   # port 5556

    while True:
      ret1, frame1 = cap1.read()
      if ret1: pub1.send(jpeg_encode(frame1))

      ret2, frame2 = cap2.read()
      if ret2: pub2.send(jpeg_encode(frame2))

      # skip frame nếu camera không kịp
      time.sleep(0.01)  # ~100fps cap
