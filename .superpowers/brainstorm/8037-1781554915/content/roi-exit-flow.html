CAM1 — Phát hiện đối tượng ra khỏi ROI

Vẽ 3-4 vùng ROI trên từng camera → phát hiện object rời khỏi zone

Luồng xử lý chi tiết

1. PyQt QThread đọc frame từ USB Camera1
↓
2. YOLOv8 detect objects (person)
↓
3. Với mỗi person detect được:
├─ Lấy center point (hoặc foot point) của bounding box
├─ Kiểm tra point có nằm trong ROI polygon nào không
└─ Nếu có → đánh dấu "inside ROI"
↓
4. Track object theo thời gian (centroid tracking / IoU tracking):
├─ Nếu frame trước: inside ROI, frame này: outside ROI
└─ → Trigger "Object LEFT zone" alert
↓
5. Gửi frame + overlay qua MediaMTX RTSP
Đồng thời gửi alert qua WebSocket → Dashboard
↓
6. Ghi violation vào SQLite

ROI Drawing UX

Cho phép vẽ đa giác (polygon) trực tiếp trên frame video real-time:

  • Click để thêm điểm, double-click để đóng polygon
  • Kéo thả điểm để điều chỉnh
  • Mỗi ROI có thể gán label: "Vùng cấm", "Khu vực nguy hiểm", "Lối ra"
  • Lưu ROI config vào JSON/DB per camera

Thuật toán "Object leaving ROI"

tracker = {}  # object_id → {"inside_roi": bool, "roi_ids": []}

Mỗi frame:
  detections = yolo(frame)
  for det in detections:
    track_id = tracker.update(det)  # IoU / centroid
    foot_point = (det.x_center, det.y2)  # đáy bbox

    inside = any(point_in_polygon(foot_point, roi) for roi in rois)

    # Transition check
    if tracker[track_id].inside_roi and not inside:
      trigger_alert(track_id, "LEFT_ROI", roi_id)

    tracker[track_id].inside_roi = inside
